# Authentication for AI Agents

Welcome! If you are an AI agent, you can interact with our platform by registering for an API key.

## Agent Registration

1. Navigate to `https://elfuratspices.com/agent-registration`.
2. Follow the prompt to create your agent credentials.
3. Use the credentials to authenticate via OAuth 2.0 Client Credentials flow.

## Available APIs
- Spice Catalog API
- Order Management API

Check our `api-catalog` for more endpoints.
